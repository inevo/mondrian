What is Mondrian?

Mondrian is an OLAP server written in Java. It enables you to interactively analyze very large datasets stored in SQL databases without writing SQL.

Read the overview, installation instructions, javadoc, and the roadmap.

See release notes.
June 28th, 2002
Mondrian to implement the JOLAP specification

JOLAP, the long-awaited J2EE API for access to OLAP servers (http://www.jcp.org/jsr/detail/69.jsp), is approaching its public review stage. Mondrian is ideally placed to become the first, and leading, JOLAP implementation.
