What is Mondrian?

Mondrian is an OLAP server written in Java. It enables you to interactively analyze very large datasets stored in SQL databases without writing SQL.

Mondrian is a component of the Pentaho BI Platform; see the FAQ.

Read the overview, installation instructions, javadoc, and the roadmap.

December 19, 2005
Mondrian 2.0 released

We are pleased announce a major new release of the Mondrian OLAP Server.

Mondrian release 2.0.1 (production) adds many new features for building scalable, powerful and flexible Business Intelligence applications on top of Open Source and Commercial databases. This release builds upon the Mondrian 1.1 release, popular for its small footprint, flexible deployment model, and implementation of the industry-standard MDX language.

Some of the added features are:

  • Aggregate tables improve the performance of large data warehouses;
  • Calculated sets defined in the schema, and WITH SET syntax to define sets within an MDX query;
  • User-defined functions;
  • Enhanced support for parent-child hierarchies;
  • Enhanced XML for Analysis (XML/A) support;
  • Enhanced support for internationalized/localized (I18N/L10N) applications;
  • Pushdown SQL. To improve performance, Mondrian automatically translates filters and aggregations into SQL which can be executed on the underlying RDBMS;
  • Support for the Apache Derby pure Java embedded RDBMS.

Mondrian is the OLAP Server component of the Pentaho open-source BI stack. See http://www.pentaho.org for more details.

Mondrian joins forces with Pentaho

ORLANDO, Fla. — (BUSINESS WIRE) — Nov. 10, 2005 — Today, Pentaho Corp., the world's leading provider of open source Business Intelligence (BI) software, announced that the Mondrian On-Line Analytical Processing (OLAP) project is joining forces with Pentaho to become a permanent feature of the Pentaho BI project, which provides comprehensive reporting, analysis, dashboards, workflow, and data mining capabilities.
