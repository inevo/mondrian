People

Mondrian is developed by Julian Hyde (project lead), Paul Dymecki, Sean McCullogh, and Andreas Voss.

Sites

Mondrian is hosted at SourceForge. It is also listed at Advogato and freshmeat.net.

The source code (including this page) is hosted in John Sichi's perforce server. Thanks to John, and to Perforce for letting us use their server gratis.

Related projects

Andreas is developing JPivot - a JSP-based pivot table, which works on top of mondrian.

MonRG is a spin-off project to generate wrapper classes for access to Java resource bundles.

Julian works on Saffron in his spare spare time.

Technology

Mondrian is in the process of implementing JOLAP and MDX specifications. It is built using Xerces, Xalan, Tomcat, JavaCUP, and Ant; and works with MySQL, Oracle, ...
