People

Mondrian is developed by Julian Hyde (project lead), Paul Dymecki, Sean McCullough, and Andreas Voss.

Sites

Mondrian is hosted at SourceForge. It is also listed at Advogato and freshmeat.net.

The source code (including this page) is hosted in Eigenbase's perforce server. Thanks to John Sichi for setting this up, and to Perforce for letting us use their server gratis.

Related projects

Andreas is developing JPivot - a JSP-based pivot table, which works on top of mondrian.

MonRG is a spin-off project to generate wrapper classes for access to Java resource bundles.

Total Information Awareness. (World Domination would be okay if we did it using Open Source. Wouldn't it?)

Julian works on Saffron and Joogle in his spare spare time.

Technology

Mondrian is in the process of implementing JOLAP and MDX specifications. It is built using Xerces, Xalan, Tomcat, JavaCUP, and Ant; and works with hsqldb, IBM DB2, Microsoft Access and SQL Server, MySQL, Oracle, PostgreSQL, ...
