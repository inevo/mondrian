To run the Mondrian unit tests:
  1. Extract all JAR files in mondrian.war to ../lib.
  2. Load the FoodMart dataset into the database of your choice. (Instructions are in the install guide.)
  3. Customize mondrian.properties, in particular mondrian.jdbcDrivers and mondrian.foodmart.jdbcURL.
  4. Run mondrian.bat (on Windows) or mondrian.sh (on UNIX)
